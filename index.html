Introduction
var helloWorld = "Hello, World!";
console.log(helloWorld.concat(" ", "Welcome to Developer Tricks!"));

My name is Matt, and I created Developer Tricks to store and display all the bits about web development that I lookup and find useful. This information is pulled from resources like Stack Overflow, CSS Tricks, YouTube, Medium, etc. and I wanted a solution to store all of the information that I find into one central location. Here it is! Everything I find useful in Web Development (for now). Each and every entry is tested and used by me, and helped me solve some kind of problem I was stuck on. These can be simple, complex or hacky solutions, as long as they work how I want them to! Clearly, this is not an exhaustive list. Feel free to contact me on Twitter with any suggestions or additions that you can think of!
